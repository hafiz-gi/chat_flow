# ChatFlow
ChatFlow is a complete, all-in-one messaging app built for seamless communication on Android and iOS. Send instant text messages, voice notes, photos, videos, documents, and links with lightning speed.

Create group chats, share live location, run polls, react to messages, and stay connected in real time. Enjoy crystal-clear audio and HD video calls with friends, family, or teams anywhere in the world.

Sign in easily using Google, Apple, Facebook, or email for a secure, hassle-free experience. With end-to-end encryption, cloud sync, push notifications, media sharing, read receipts, typing indicators, and a clean, modern interface, ChatFlow keeps every conversation smooth and private.
